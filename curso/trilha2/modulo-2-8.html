‚ö° Por que Performance Importa

Performance e experiencia estao diretamente conectadas. Animacoes lentas, travamentos e stuttering destroem completamente a imersao do usuario. O objetivo e sempre atingir 60fps - isso significa que cada frame deve ser renderizado em no maximo 16.67ms.

üéØ Conceitos Fundamentais

  • ‚Ä¢ Animacoes lentas: Criam uma experiencia ruim e frustrante
  • ‚Ä¢ 60fps e o objetivo: Cada frame deve renderizar em 16.67ms
  • ‚Ä¢ Jank e stuttering: Travamentos que destroem a imersao
  • ‚Ä¢ Frame budget: Voce tem apenas 16ms para calcular, renderizar e pintar

üìä Dados Importantes

53% dos usuarios abandonam sites que demoram mais de 3 segundos para carregar ou que apresentam animacoes travadas.

üéØ Propriedades Performaticas

Nem todas as propriedades CSS sao iguais quando se trata de performance. Algumas sao aceleradas por GPU e extremamente rapidas, enquanto outras causam reflow e repaint custosos.

‚úì USAR (GPU-Accelerated)

  • ‚úì transform - translate, scale, rotate
  • ‚úì opacity - fade in/out
  • ‚úì will-change - preparar GPU
  • ‚úì contain - isolamento de layout

‚úó EVITAR (Causam Reflow)

  • ‚úó width / height
  • ‚úó margin / padding
  • ‚úó top / left / right / bottom
  • ‚úó font-size - recalcula layout
